## Cara Pakai

1. Buka [RifqyNotes](https://rydevs29.github.io/RifqyNotes)
2. Masukkan **email & password** → klik **Masuk**
3. Klik **Rekam Suara** → bicara bebas
4. Klik **Berhenti** → **AI Ringkas** → **Simpan**
5. Catatan tersimpan **offline di browser**
6. Klik **Logout** → data tetap aman
